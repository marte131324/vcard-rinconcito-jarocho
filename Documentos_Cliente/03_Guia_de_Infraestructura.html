2. Enrutamiento y Dominio (Vercel)

La tarjeta se ha desplegado en Edge Networks a través de Vercel (San Francisco) con certificación SSL/TLS automática para seguridad en los pagos y visitas.

  • URL Actual: https://vcard-rinconcito-jarocho.vercel.app
  • Tiempo de respuesta global optimizado para evitar latencia al momento del splash o carga del sitio.

3. Seguridad y Archivos

Dado que no se depende de un servicio SQL de base de datos intermedio, el sistema "App.js" y "Admin" corren localmente sobre el navegador de su dispositivo móvil, haciéndolo increíblemente veloz e inmune a vulnerabilidades cloud convencionales.

Cualquier cambio de número telefónico de recepción (WhatsApp) requerirá notificación al equipo de desarrollo para cambiar la variable WHATSAPP_NUMBER en la base de código.
